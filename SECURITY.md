## Reporting a Security Vulnerability

SurveySentinel takes security seriously.

If you believe you have discovered a security vulnerability within any SurveySentinel software, documentation or associated infrastructure, please report it responsibly.

Please do **not** disclose vulnerabilities publicly until they have been investigated.

## Please Include

- Product or repository affected
- Version number
- Detailed description
- Steps to reproduce
- Potential impact
- Proof of concept (if appropriate)

## Response Targets

| Stage | Target |
|--------|---------|
| Initial acknowledgement | 5 working days |
| Investigation | 30 days |
| Resolution | As appropriate |

---

## Responsible Disclosure

SurveySentinel supports coordinated vulnerability disclosure and requests that researchers allow reasonable time for investigation before public disclosure.
