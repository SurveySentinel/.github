# Contributing

Thank you for your interest in SurveySentinel.

At present, SurveySentinel is developed by the core development team.

Public contributions are not currently accepted while the platform remains under active commercial development.

---

## Feedback

Constructive feedback is always welcome.

Suggestions for:

- documentation
- usability
- compatibility
- bug reports
- feature ideas

may be submitted through GitHub Issues.

---

## Code Contributions

Pull requests containing source code are currently disabled.

As SurveySentinel matures, selected repositories may become open to community contributions.

---

Thank you for supporting SurveySentinel.
